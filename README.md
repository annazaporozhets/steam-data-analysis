# Steam Exploratory Data Analysis

## Description
This project focuses on conducting an Exploratory Data Analysis (EDA) on a comprehensive dataset of video games released on the Steam platform. The primary goal was to understand market dynamics, uncover relationships between game pricing and user feedback, and analyze factors that influence player engagement.


## Key Insights & Findings
* Pricing Trends: Analysis of the dataset revealed that the majority of successful titles are distributed in the lower price segments, which significantly correlates with higher ownership and playtimes.
* Player Engagement: I explored the impact of achievements on player retention, showing that games offering achievement systems generally encourage longer average playtimes.
* Market Activity: Through temporal analysis, I identified key periods of industry growth and release activity.

## Technologies Used
* Language: Python
* Libraries: Pandas & NumPy for data manipulation and cleaning. Seaborn & Matplotlib for statistical data visualization.
* Environment: Jupyter Notebook (Google Colab).
